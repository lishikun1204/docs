## 一、Jenkins 平台介绍

### 1.1 Jenkins 概述

Jenkins 是一个开源的自动化构建工具，用于实现持续集成和持续部署（CI/CD）。它支持多种构建类型，包括自由风格项目、流水线项目、多配置项目等。

### 1.2 核心功能

| 功能 | 说明 |
|------|------|
| 持续集成 | 自动构建、测试代码 |
| 持续部署 | 自动部署应用 |
| 流水线 | 可视化构建流程 |
| 插件生态 | 丰富的插件扩展 |
| 分布式构建 | 多节点并行构建 |
| 构建历史 | 完整的构建记录 |

### 1.3 界面介绍

**主要界面元素：**

- **导航栏**：包含新建任务、管理 Jenkins 等选项
- **构建队列**：显示等待执行的构建任务
- **执行器状态**：显示当前运行的构建任务
- **项目列表**：显示所有 Jenkins 项目
- **系统消息**：显示系统通知和告警

### 1.4 术语定义

| 术语 | 说明 |
|------|------|
| 项目 | Jenkins 中的构建任务 |
| 构建 | 执行一次构建过程 |
| 流水线 | 定义构建、测试、部署的流程 |
| 执行器 | 执行构建任务的进程 |
| 节点 | 执行构建任务的服务器 |
| 插件 | 扩展 Jenkins 功能的组件 |
| 凭据 | 存储认证信息的安全对象 |
| 工作空间 | 构建任务的工作目录 |

## 二、用户登录与权限

### 2.1 登录 Jenkins

1. **访问 Jenkins**：打开浏览器，输入 Jenkins URL（如 `http://jenkins.example.com`）
2. **输入凭据**：在登录页面输入用户名和密码
3. **点击登录**：点击 "登录" 按钮

### 2.2 用户注册

1. **访问注册页面**：点击登录页面的 "注册" 链接
2. **填写信息**：输入用户名、密码、邮箱
3. **提交注册**：点击 "注册" 按钮
4. **等待审批**：管理员审核后激活账户

### 2.3 忘记密码

1. **访问登录页面**：打开 Jenkins 登录页面
2. **点击 "忘记密码"**：在登录页面点击 "忘记密码" 链接
3. **输入邮箱**：填写注册时使用的邮箱
4. **重置密码**：按照邮件提示重置密码

### 2.4 权限管理

**权限级别：**

| 权限 | 说明 |
|------|------|
| 管理员 | 完全控制权限 |
| 项目管理员 | 管理项目权限 |
| 开发者 | 构建和查看项目权限 |
| 查看者 | 只读权限 |

**权限分配：**

1. 管理员登录 Jenkins
2. 进入 "管理 Jenkins" → "管理用户"
3. 选择用户，点击 "配置"
4. 在 "权限" 部分设置用户权限
5. 点击 "保存"

## 四、构建配置详解

### 4.1 自由风格项目配置

**源码管理：**

```bash
# Git 配置
Repository URL: https://github.com/username/project.git
Credentials: 选择 Git 凭据
Branch Specifier: */main
```

**构建步骤：**

```bash
# 执行 Shell
#!/bin/bash

# 安装依赖
npm install

# 构建项目
npm run build

# 运行测试
npm test
```

**构建后操作：**

```bash
# 归档构建产物
Files to archive: dist/**/*

# 发布测试结果
Test report XMLs: test-results/**/*.xml

# 发送邮件通知
Recipients: user@example.com
```

### 4.2 流水线项目配置

**流水线脚本：**

```groovy
pipeline {
    agent any
    
    parameters {
        string(name: 'ENVIRONMENT', defaultValue: 'staging', description: '部署环境')
    }
    
    stages {
        stage('构建') {
            steps {
                checkout scm
                sh 'npm install'
                sh 'npm run build'
            }
        }
        
        stage('测试') {
            steps {
                sh 'npm test'
            }
        }
        
        stage('部署') {
            steps {
                sh "npm run deploy -- --env=${params.ENVIRONMENT}"
            }
        }
    }
    
    post {
        success {
            echo '构建成功！'
        }
        failure {
            echo '构建失败！'
        }
    }
}
```

**流水线语法生成器：**

1. 进入流水线项目配置页面
2. 点击 "流水线语法" 链接
3. 选择步骤类型
4. 填写参数
5. 点击 "生成流水线脚本"
6. 复制生成的脚本到流水线配置

### 4.3 多配置项目配置

**构建矩阵：**

1. 进入项目配置页面
2. 点击 "添加轴"
3. 选择轴类型：
   - **用户定义的轴**：自定义参数
   - **标签轴**：按节点标签
   - **JDK 轴**：按 JDK 版本
4. 配置轴值
5. 点击 "保存"

**示例：**

```bash
# 构建矩阵配置
轴名称：BROWSER
轴值：chrome, firefox, safari

轴名称：OS
轴值：windows, linux, mac
```

## 六、构建触发方式

### 6.1 手动触发

1. 进入项目页面
2. 点击左侧的 "立即构建" 链接
3. （可选）填写构建参数
4. 点击 "开始构建"

### 6.2 定时构建

**配置定时构建：**

1. 进入项目配置页面
2. 找到 "构建触发器" 部分
3. 勾选 "定期构建"
4. 填写 cron 表达式
5. 点击 "保存"

**Cron 表达式：**

```
# 格式：分 时 日 月 周

# 示例：
0 2 * * *     # 每天凌晨 2 点
0 12 * * 1-5  # 工作日中午 12 点
*/30 * * * *  # 每 30 分钟
0 */2 * * *   # 每 2 小时
```

### 6.3 SCM 变更触发

**配置 SCM 变更触发：**

1. 进入项目配置页面
2. 找到 "构建触发器" 部分
3. 勾选 "轮询 SCM"
4. 填写 cron 表达式
5. 点击 "保存"

**Git 钩子触发：**

1. 进入项目配置页面
2. 找到 "构建触发器" 部分
3. 勾选 "GitHub hook trigger for GITScm polling"
4. 在 Git 仓库中配置 webhook
5. 点击 "保存"

### 6.4 其他项目触发

**配置依赖构建：**

1. 进入项目配置页面
2. 找到 "构建触发器" 部分
3. 勾选 "在其他项目构建后触发"
4. 输入依赖项目名称
5. 选择触发条件（成功、失败、不稳定）
6. 点击 "保存"

### 6.5 远程触发

**配置远程触发：**

1. 进入项目配置页面
2. 找到 "构建触发器" 部分
3. 勾选 "触发远程构建"
4. 设置认证令牌
5. 点击 "保存"

**远程触发 URL：**

```
http://jenkins.example.com/job/project-name/build?token=TOKEN_NAME
```

## 九、流水线配置

### 9.1 流水线语法

**基本结构：**

```groovy
pipeline {
    agent any
    
    parameters {
        // 构建参数
    }
    
    environment {
        // 环境变量
    }
    
    stages {
        stage('阶段1') {
            steps {
                // 执行步骤
            }
        }
        
        stage('阶段2') {
            steps {
                // 执行步骤
            }
        }
    }
    
    post {
        // 构建后操作
    }
}
```

**常用步骤：**

| 步骤 | 说明 |
|------|------|
| checkout | 检出代码 |
| sh | 执行 Shell 命令 |
| bat | 执行 Windows 命令 |
| echo | 输出信息 |
| archiveArtifacts | 归档构建产物 |
| junit | 发布测试结果 |
| sshPublisher | SSH 发布 |
| docker | Docker 操作 |

### 9.2 流水线示例

**Java 项目构建：**

```groovy
pipeline {
    agent any
    
    tools {
        maven 'Maven 3.8.6'
        jdk 'JDK 11'
    }
    
    stages {
        stage('检出代码') {
            steps {
                checkout scm
            }
        }
        
        stage('构建') {
            steps {
                sh 'mvn clean package -DskipTests'
            }
        }
        
        stage('测试') {
            steps {
                sh 'mvn test'
            }
        }
        
        stage('部署') {
            steps {
                sh 'mvn deploy'
            }
        }
    }
    
    post {
        success {
            echo '构建成功！'
        }
        failure {
            echo '构建失败！'
        }
    }
}
```

**前端项目构建：**

```groovy
pipeline {
    agent any
    
    tools {
        nodejs 'Node 16'
    }
    
    stages {
        stage('检出代码') {

        }

                sh 'npm install'
            }
        }

    }

```

### 9.3 流水线最佳实践

**代码风格：**

- 使用有意义的阶段名称
- 保持步骤简洁明了
- 使用注释说明复杂逻辑

**错误处理：**

- 使用 try-catch 捕获异常
- 在 post 部分处理构建结果
- 配置适当的超时时间

**参数化构建：**

- 使用 parameters 定义构建参数
- 在流水线中使用 params 访问参数
- 为参数设置默认值和描述

**环境变量：**

- 使用 environment 定义环境变量
- 使用 withEnv 临时设置环境变量
- 避免硬编码敏感信息

## 十、常见操作问题与解决方案

### 10.1 登录问题

**无法登录：**

- **原因**：用户名或密码错误、账户被锁定、Jenkins 服务异常
- **解决方案**：
  - 检查用户名和密码
  - 联系管理员解锁账户
  - 检查 Jenkins 服务状态

**权限不足：**

- **原因**：用户权限不足、权限配置错误
- **解决方案**：
  - 联系管理员提升权限
  - 检查权限配置

### 10.2 项目创建问题

**创建失败：**

- **原因**：权限不足、项目名称已存在、配置错误
- **解决方案**：
  - 检查用户权限
  - 使用唯一的项目名称
  - 检查配置参数

**配置错误：**

- **原因**：参数填写错误、路径不存在、凭据无效
- **解决方案**：
  - 检查配置参数
  - 确认路径存在
  - 验证凭据有效性

### 10.3 构建问题

**构建失败：**

- **原因**：代码错误、依赖问题、构建脚本错误
- **解决方案**：
  - 查看控制台输出
  - 检查代码错误
  - 检查依赖配置
  - 修复构建脚本

**构建超时：**

- **原因**：构建时间过长、网络问题、资源不足
- **解决方案**：
  - 设置构建超时时间
  - 优化构建脚本
  - 检查网络连接
  - 增加构建资源

**构建卡住：**

- **原因**：死锁、无限循环、资源争用
- **解决方案**：
  - 取消构建
  - 检查构建脚本
  - 检查资源使用

### 10.4 插件问题

**插件安装失败：**

- **原因**：网络问题、版本不兼容、磁盘空间不足
- **解决方案**：
  - 检查网络连接
  - 检查 Jenkins 版本
  - 检查磁盘空间
  - 手动下载插件

**插件冲突：**

- **原因**：多个插件功能冲突
- **解决方案**：
  - 禁用冲突插件
  - 更新插件版本
  - 替换插件

### 10.5 权限问题

**访问被拒绝：**

- **原因**：权限不足、IP 限制、CSRF 保护
- **解决方案**：
  - 联系管理员提升权限
  - 检查 IP 限制
  - 重新登录获取 CSRF 令牌

**权限配置错误：**

- **原因**：权限设置错误、角色配置错误
- **解决方案**：
  - 检查权限配置
  - 检查角色分配
  - 联系管理员修复
